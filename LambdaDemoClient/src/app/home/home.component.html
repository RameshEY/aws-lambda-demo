AWS S3 and Lambda Demo

This demo app allows you to upload image files to an S3 bucket and then view the details of those images. When a file is uploaded to the specified bucket, an AWS lambda function is triggered. It retrieves the uploaded image from S3 and saves the image meta data to DynamoDB. When a file is deleted from the bucket, the same lambda fucntion is called, but this time it deletes the associated image meta data from DynamoDB. As well as allowing you to upload and delete images, this Angular client periodically calls a lambda function (via API Gateway) to retrieve all image details from DynamoDB.
